# SRHInformativeLabel

This is an Umbraco property editor that can be configured to provide info to content authors. 
Use v0 for Umbraco 7
Use v1 for Umbraco 8

## Setup

### Install Dependencies

```bash
npm install -g grunt-cli
npm install
```

### Build

```bash
grunt
```

### Watch

```bash
grunt watch
```
